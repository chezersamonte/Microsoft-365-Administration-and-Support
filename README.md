# Microsoft 365 Administration and Support

A hands-on learning portfolio focused on Microsoft 365 administration, Level 1 IT support, and day-to-day Microsoft 365 administrative tasks. This project documents my learning journey through practical labs, configurations, and real-world administration scenarios.

## Live Website

🌐 **View the portfolio here:**
https://chezersamonte.github.io/Microsoft-365-Administration-and-Support/

## Overview

This portfolio is designed to strengthen practical Microsoft 365 administration skills commonly required for Help Desk, Service Desk, IT Support, and Microsoft 365 Administrator roles.

The project covers the Microsoft 365 Admin Center and introduces the most common administrative tasks performed by Level 1 support engineers.
