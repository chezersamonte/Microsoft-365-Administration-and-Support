Learn the essential Microsoft 365 user administration tasks used by Level 1 IT Support, including user lifecycle management, password administration, administrator roles, guest users, and bulk user operations.

Locate user accounts, search by name or email address, and filter users based on account status, licenses, or other criteria.

Create User

Create a new Microsoft 365 user account and configure the username, license, password, and basic account settings.

Create Multiple Users

Quickly create several Microsoft 365 user accounts using the built-in multiple user creation wizard.

Bulk Create Users Using CSV

Import multiple users from a CSV file to simplify onboarding large numbers of employees.

Delete User or Multiple Users

Remove individual or multiple user accounts while preserving the ability to restore them during the retention period.

Recover recently deleted Microsoft 365 user accounts together with their mailbox, OneDrive, and assigned licenses when applicable.

Reset User Passwords

Reset forgotten or compromised passwords and provide users with temporary sign-in credentials.

Require users to create a new password the next time they sign in to Microsoft 365.

Block Sign-in

Temporarily prevent a user from accessing Microsoft 365 services without deleting the account.

Unblock Sign-in

Restore access for users after resolving account, security, or administrative issues.
